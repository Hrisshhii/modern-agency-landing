# Assignment UI

A responsive React web page built from a Figma design.

## Tech Stack

* React (Vite + TypeScript)
* Tailwind CSS v4

## Features

* Pixel-perfect UI based on Figma
* Fully responsive (mobile, tablet, desktop)
* Reusable components
* Clean and structured code

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
